# ParkSmart - Modern Parking Management System

A premium, modern parking management system built with HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- **Modern UI**: Dark mode with glassmorphism and smooth animations.
- **User Roles**: Separate dashboards for Users and Admins.
- **Real-time Slots**: Dynamic slot status (Available/Occupied).
- **Booking System**: Easy slot reservation for users.
- **Fare Calculation**: Automatic billing based on hours parked.
- **Admin Analytics**: Revenue tracking and slot monitoring.

## Setup Instructions
1. Make sure **XAMPP** is running (Apache and MySQL).
2. The database is already initialized if you used the setup script.
3. URL: `http://localhost/park/index.php`
4. **Admin Credentials**:
   - Email: `admin@park.com`
   - Password: `admin123`

## Technology Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript
- **Backend**: PHP (PDO)
- **Database**: MySQL
